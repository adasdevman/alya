HubSpot CRM

L'intégration avec HubSpot vous permet de :

  • Créer des contacts directement depuis le chat
  • Synchroniser les informations client
  • Gérer vos relations client de manière efficace
Comment configurer HubSpot :
  1. Cliquez sur "Intégrations" dans le menu
  2. Sélectionnez HubSpot CRM
  3. Suivez le processus d'authentification

Problèmes courants :
  • Connexion HubSpot perdue - Reconnectez-vous via le menu Intégrations
  • Messages non envoyés - Vérifiez votre connexion internet
  • Historique non visible - Actualisez la page

Pour toute autre assistance, contactez le support : support@alya.ai
